# NavTrace
Projeto - Tópicos Integradores
Projeto NavTrace

NavTrace é uma plataforma web para democratizar o monitoramento logístico de embarcações para pequenas empresas e varejistas. O sistema permite rastreamento, análise de risco e visualização em tempo real de rotas, atrasos e eventos críticos.

Objetivo
Permitir que empresas que não possuem acesso a sistemas caros de monitoramento tenham visibilidade sobre suas entregas marítimas.

Problema
Pequenas transportadoras e varejistas possuem pouca visibilidade sobre o transporte de suas cargas, gerando perdas financeiras, atrasos e falhas de comunicação.

Solução
NavTrace oferece:
– rastreamento de embarcações
– visualização de rotas
– alertas de risco
– histórico de transporte
– análise de desempenho logístico

Tecnologias
Backend: Python, FastAPI
Banco de dados: PostgreSQL
ORM: SQLAlchemy
Frontend: React

Status
MVP em desenvolvimento.
